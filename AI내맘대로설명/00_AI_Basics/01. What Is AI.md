# 01. What Is AI

# AI는 무엇인가?

많은 사람들이 AI를 보면 이렇게 생각한다.

- 사람처럼 생각한다
    
- 스스로 판단한다
    
- 기억한다
    
- 이해한다
    

하지만 실제 AI는 우리가 생각하는 것과 조금 다르게 동작한다.

---

# AI는 "확률 기반 예측 시스템"에 가깝다

현재 대부분의 AI는 다음과 같은 방식으로 동작한다.

```text
입력
↓
과거에 학습한 데이터 패턴 분석
↓
가장 가능성이 높은 결과 생성
```

예를 들어:

```text
오늘 날씨가 너무
```

라는 문장이 있으면 사람은 자연스럽게:

- 좋다
    
- 춥다
    
- 덥다
    

같은 단어를 예상한다.

AI도 비슷하다.

인터넷의 엄청난 문서를 학습하면서:

```text
"오늘 날씨가 너무" 다음에는
"좋다"가 자주 나왔네?
```

같은 패턴을 학습한다.

즉 AI는:

```text
다음에 무엇이 나올 확률이 높은가
```

를 계산하는 시스템에 가깝다.

---

# 그런데 왜 사람처럼 보일까?

AI는 정말 엄청난 양의 데이터를 학습한다.

예를 들어:

- 책
    
- 뉴스
    
- 블로그
    
- 코드
    
- 논문
    
- 대화 데이터
    

같은 것들을 매우 많이 학습한다.

그래서 질문을 받으면:

```text
비슷한 패턴을 찾고
↓
그럴듯한 답변을 생성
```

하게 된다.

이 과정이 자연스럽게 보이기 때문에 사람처럼 느껴지는 것이다.

---

# AI는 "이해"와는 조금 다르다

사람은:

- 경험
    
- 감정
    
- 상황
    
- 의도
    

를 함께 고려한다.

하지만 AI는 기본적으로:

```text
패턴과 확률
```

을 기반으로 동작한다.

그래서 가끔:

- 틀린 말을 자신 있게 하거나
    
- 존재하지 않는 정보를 만들어내거나
    
- 엉뚱한 답을 하는 경우
    

도 발생한다.

이를 보통:

```text
Hallucination (환각)
```

이라고 부른다.

---

# AI가 잘하는 것

AI는 특히 다음 분야에 강하다.

- 요약
    
- 문서 정리
    
- 번역
    
- 코드 생성
    
- 반복 업무
    
- 검색 보조
    
- 문장 생성
    

즉:

```text
대량의 정보를 빠르게 처리하는 작업
```

에 매우 강하다.

---

# AI가 약한 것

반대로 다음 영역은 아직 약하다.

- 완벽한 사실 판단
    
- 책임 있는 결정
    
- 긴 논리 일관성 유지
    
- 최신 정보 보장
    
- 회사 내부 상황 이해
    

그래서 실제 업무에서는:

```text
AI + 사람 검토
```

구조를 많이 사용한다.

---

# 현재 AI의 핵심 특징

현재 AI를 가장 간단하게 표현하면:

```text
엄청난 데이터를 학습한
확률 기반 언어 생성 시스템
```

정도로 이해할 수 있다.

그리고 이후에는:

- LLM
    
- 벡터
    
- RAG
    
- AI Agent
    
- Guardrail
    

같은 개념들이 이 위에 추가로 등장하게 된다.
